# URBAN NEST | 都市型賃貸マンション仲介サービス

不動産仲介サービスのランディングページです。ポートフォリオ作品として制作しました。

🔗 **Live Site:** https://kkato0219.github.io/urban-nest/

---

## 使用技術

| 技術 | 用途 |
|------|------|
| HTML5 | セマンティックなマークアップ |
| CSS3 | レスポンシブデザイン・アニメーション |
| JavaScript | インタラクション・バリデーション |
| Typed.js | タイピングアニメーション |
| particles.js | パーティクル背景 |
| Swiper.js | スライダー |
| AOS.js | スクロールアニメーション |
| CountUp.js | 数字カウントアップ |
